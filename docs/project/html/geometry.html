Cuadrado

¿Qué es un cuadrado?

Un cuadrado es un polígono de 4 lados iguales y 4 ángulos rectos (90°).

Perímetro: P = lado × 4
Área: A = lado²

Introduce la longitud del lado en centímetros para calcular el perímetro y el área.

Triángulo

¿Qué es un triángulo?

Un triángulo es un polígono de 3 lados. Para calcular su área necesitas la base y la altura.

Perímetro: P = lado1 + lado2 + base
Área: A = (base × altura) / 2

Introduce la longitud de los lados en centímetros para calcular el perímetro y el área.

Triángulo Isósceles (Altura)

Triángulo Isósceles

Un triángulo isósceles tiene 2 lados iguales y 1 diferente. Podemos calcular su altura con los lados.

Altura: h = √(lado² − base²/4)

Triángulo Escaleno (Altura)

Triángulo Escaleno

Un triángulo escaleno tiene todos sus lados diferentes. Usamos la fórmula de Herón para calcular el área y luego la altura.

Semiperímetro: s = (a + b + c) / 2
Área (Herón): A = √[s(s-a)(s-b)(s-c)]
Altura: h = (2 × A) / base

Círculo

¿Qué es un círculo?

Un círculo es el conjunto de puntos equidistantes de un centro. El radio es la distancia del centro al borde.

Diámetro: d = radio × 2
Circunferencia: C = diámetro × π
Área: A = radio² × π

Introduce el radio del círculo para calcular la circunferencia y el área.
